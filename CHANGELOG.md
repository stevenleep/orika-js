# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.2.1] - 2024-11-14

### Added
- Complete TypeScript object mapping functionality
- Support for PO/DTO/VO transformations
- Built-in converters for common data types
- Async mapping with parallel processing
- Collection support (Array, Map, Set)
- Validation hooks
- Custom converters support
- Decorator-based mapping configuration
- Bidirectional mapping
- Mapping statistics and caching
- Expression evaluator for complex transformations
- Type-safe access utilities
- Batch processing capabilities

### Features
- **Type Safety**: Full TypeScript generics support
- **Auto Mapping**: Automatic mapping for same-name fields
- **Flexible Configuration**: MapperConfigBuilder for fluent API
- **Web Support**: Full browser environment compatibility
- **Performance**: Caching and optimization features

### Documentation
- Comprehensive README with examples
- Multiple example files demonstrating different use cases
- Web examples with interactive demos
